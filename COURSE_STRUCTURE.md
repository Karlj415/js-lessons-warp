## How to Use This Course

1. **Start with the main README.md** - Get an overview of the entire course
2. **Follow lessons in order** - Each builds on previous knowledge
3. **Read each lesson's README** - They contain all the content
4. **Type the code yourself** - Don't copy-paste!
5. **Complete the challenges** - They reinforce learning
6. **Take your time** - Understanding > Speed

## Recommended Learning Path

### Phase 1: JavaScript Fundamentals (2-3 weeks)
- Lessons 00-02: Setup and basics
- Build: Simple calculator, FizzBuzz

### Phase 2: Interactive Web Pages (2-3 weeks)
- Lessons 03-04: DOM manipulation and async
- Build: Todo list, GitHub profile viewer

### Phase 3: Modern JavaScript (1-2 weeks)
- Lessons 05-06: ES6+ features and testing
- Build: Refactored projects with tests

### Phase 4: React & Front-End (3-4 weeks)
- Lessons 07-10: React, state management, tools
- Build: Movie search app, Redux todo app

### Phase 5: Back-End Development (4-5 weeks)
- Lessons 11-15: Node.js, Express, databases, auth, testing
- Build: Complete blog API with authentication

### Phase 6: Deployment & Advanced (2-3 weeks)
- Lessons 16-18: Deployment, advanced topics, capstone
- Build: Full-stack event planner app
